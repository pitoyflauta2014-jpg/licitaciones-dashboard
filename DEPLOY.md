# Publicar en GitHub Pages

Archivos que deben subirse al repositorio:

- `index.html`
- `assets/ng-logo.png`
- `README.md`
- `.nojekyll`

Pasos:

1. Abrir el repositorio `licitaciones-dashboard` en GitHub.
2. Elegir `uploading an existing file`.
3. Arrastrar el contenido de esta carpeta `public_site`.
4. Confirmar con `Commit changes`.
5. Ir a `Settings > Pages`.
6. En `Build and deployment`, elegir `Deploy from a branch`.
7. Seleccionar branch `main` y carpeta `/root`.
8. Guardar.

GitHub publicará el sitio en una URL similar a:

`https://pitoyflauta2014-jpg.github.io/licitaciones-dashboard/`

Si el repositorio está privado y GitHub no deja activar Pages, cambiarlo a público o usar Streamlit Cloud.
